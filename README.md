# 📊 منصة تحليلات الموارد البشرية بالذكاء الاصطناعي
# HR Analytics AI Platform - رسال الود لتقنية المعلومات

## 🚀 التشغيل في 3 خطوات

### الخطوة 1: تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### الخطوة 2: تشغيل التطبيق
```bash
streamlit run hr_analytics_app.py
```

### الخطوة 3: رفع البيانات
- افتح المتصفح على الرابط الذي يظهر (عادة http://localhost:8501)
- ارفع ملف "بيانات-موظفين-تجريبية-رسال-الود.xlsx" من القائمة الجانبية
- تنقل بين الصفحات واستمتع بالتحليلات!

## 📁 ملفات المشروع

| الملف | الوصف |
|-------|-------|
| `hr_analytics_app.py` | التطبيق الكامل (الملف الرئيسي) |
| `بيانات-موظفين-تجريبية-رسال-الود.xlsx` | بيانات تجريبية لـ 50 موظف |
| `requirements.txt` | المكتبات المطلوبة |

## 📌 صفحات التطبيق

1. **🏠 نظرة عامة** - KPIs + رسوم بيانية + رؤى ذكية تلقائية
2. **💰 الرواتب والتكاليف** - تحليل مفصل بالريال (راتب + سكن + نقل + تأمينات + طبي)
3. **🔄 الدوران الوظيفي** - تتبع الاستقالات + أسباب المغادرة + تنبيهات
4. **⚡ الأداء والإنتاجية** - Radar Chart + ترتيب الموظفين + علاقة الراتب بالأداء
5. **👥 التوظيف والاستقطاب** - قمع التوظيف + مصادر الاستقطاب + أيام التوظيف
6. **🤖 المحلل الذكي** - رؤى تلقائية + أسئلة مباشرة بالذكاء الاصطناعي
7. **📋 بيانات الموظفين** - عرض + بحث + تصفية
8. **📥 تصدير التقارير** - تحميل تقارير مخصصة بصيغة Excel

## 🤖 تفعيل الذكاء الاصطناعي (اختياري)
لتفعيل المحلل الذكي المتقدم مع Claude:
```bash
export ANTHROPIC_API_KEY="sk-ant-xxxxx"
```
التطبيق يعمل بدون API key مع تحليل محلي أساسي.
